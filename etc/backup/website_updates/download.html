Instructions for Installing MOST


After installing MOST, if the error message below is displayed when starting up MOST:

click OK to close the error message and start MOST. When the program appears, click Options -> Set GLPK Properties (Alt+O then Alt+L), then click the "Reset to Defaults" button (Alt+R) then click the OK button (Alt+O or Enter). The error message will no longer be displayed.

If installer does not work due to Windows protections or is blocked by other program, the Executable Jar zip file can be downloaded. Extract the zip into a folder and double-click the MOST.jar file to run MOST. Windows 8 is known to have issues with using the installer.


Java Version 7 must be installed in order to run MOST. Java can be installed at java.com/download. This is unnecessary for Macintosh since Java is included in the installation package.

How to determine if Java is installed and what version:

Windows 7 - Control Panel:

Change the View Category to either Large Icons or Small Icons and locate the Java icon. (If there is no Java icon, Java is not installed.) Click on the icon or the word "Java" and then click on the About button. If the About Java box says "Version 7", the correct version of Java is installed.

Windows XP - Control Panel:

Same as above but no need to change the View.

Windows 7 or XP - Command Line:

Open a command window and type:
java -version
The output will look something like this:
java version "1.7.0_45"
If the version starts with 1.7 the correct version of Java is installed.

Linux:

Open a terminal and type:
java -version
The output will look something like this:
java version "1.7.0_45"
If the version starts with 1.7 the correct version of Java is installed.
