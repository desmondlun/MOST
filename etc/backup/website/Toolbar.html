Toolbar

Toolbar with Save, Copy, Paste, Undo, Redo, and Find buttons.

Save:

See Model Menu - Saving Files.

Copy/Paste:

The Paste button will be grayed out if the cell(s) selected are in a non-editable column, or an optimization results table.

Undo/Redo:

These buttons will be grayed out if there are no edits to undo or redo, or if an optimization results table is selected. Any edit will clear all redo events and the button will become grayed out. The arrows next to the buttons will each open a menu which contains all available undo or redo events respectively.

Find/Replace:

This button opens the Find/Replace dialog.
