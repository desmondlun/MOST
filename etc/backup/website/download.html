E. coli consensus model (Smallbone, 2013) has been added to the Model Collection. A few reactions and metabolites have been added to the visualization database.

You must have Java version 1.7 or greater installed on your computer before running MOST. Click here for instructions on determining if Java is installed. Java can be downloaded from this website.

MOST executable jar can be used for Windows and Macintosh. Extract the zip into a folder and double-click the MOST.jar file to run MOST. To use the Ipopt solver with Macintosh, GFortran needs to be installed. See Instructions for Installing MOST for further instructions on setting up your system to run MOST.

Gurobi 6.5.2 is the minimum version of Gurobi that can be used by MOST Version 2.0 alpha-6 and later releases. Click here for instructions on setting up Gurobi on Windows.


If the error message below is displayed when starting up MOST:

click OK to close the error message and start MOST. When the program appears, click Options -> Set GLPK Properties (Alt+O then Alt+L), then click the "Reset to Defaults" button (Alt+R) then click the OK button (Alt+O or Enter). The error message will no longer be displayed.

If a message that starts with "Click Options to select other solvers and suppress Gurobi solver error below." appears in the console when MOST is launched, the most likely cause is that Gurobi is installed but the license is expired. Gurobi will not work. Please select other solvers in the Select Solvers dialog.
