Keyboard Shortcuts

List of Keyboard Shortcuts:

Ctrl+A = Select All (Does not include column names. See Select All.
Ctrl+C = Copy
Ctrl+E = Clear Contents
Ctrl+F = Find
Ctrl+S = Save
Ctrl+V = Paste
Ctrl+Y = Redo
Ctrl+Z = Undo
These shortcuts will work in the selected component if it is a legal operation for that component. Context Menus will show available shortcuts for menu items.
