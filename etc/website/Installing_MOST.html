Installing MOST

Requirements:

Java JRE 1.7. java.com/download.

To determine if Java is installed and what version is installed, visit the Software page.

Gurobi jar file and license. www.gurobi.com if FBA or GDBB are to be run.


Windows:

Double-click on MOST_Setup.exe and click the "Next" button:
Click the "I Agree" button on the license dialog:
Choose a destination folder or use the default, and click the "Next" button:
Click the "Install" button:
After installation is complete, click the "Finish" button. MOST has been successfully installed.

Gurobi installation instructions can be found here.
In order to run Gurobi, you also need to install the Gurobi JNI library. See topic #42 on the Gurobi FAQ's page for installation instructions.

Known Issues:

Windows 8 is known to have issues with using the installer. If the installer will not work, download the MOST Executable Jar and extract it into a directory, then double-click on the MOST.jar file to run MOST.

Linux:

Download MOST for Linux, and extract the gzip compressed file. MOST can be run from the command line using the java command:
java -jar MOST.jar
in the directory where the gzip was uncompressed.

Gurobi installation instructions can be found here.
In order to run Gurobi, you also need to install the Gurobi JNI library. See topic #42 on the Gurobi FAQ's page for installation instructions.
Once Gurobi is set up correctly, the command to run MOST must include the library path.
java -Djava.library.path=<JNI directory> -jar MOST.jar
where JNI directory is the directory that contains libGurobiJniXX.so file. "XX" is the version number.
